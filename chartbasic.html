Installation

After downloading the libraries from here, the setup is documented midway down the page in the Getting Started section. RazorFlow Components

All visual representations of data are distingued as five categories, referenced as Components. You can pick any combination of these to add to a Dashboard depending on your needs.This heiarchy was tricky for me to understand at first.
Components are added to Dashboards, not the other way around. If you thought of the Dashboard as a pizza, components would be the toppings.

  • Chart Component
    • Just as it sounds. Pie, Bar Charts, Etc.

  • Form Component
    • User input forms such as Text, Number, & Dropdowns.

  • Guage Component
    • Measuring one piece of data. An example would be 'Total Monthly Sales';

  • KPI Component
    • "Key Performance Indicators" - show current values of multiple important metrics.

  • Table Component
    • Formatted tables. This could be used to show records in a database.
