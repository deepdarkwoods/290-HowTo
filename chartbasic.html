After downloading the libraries from here, the setup is documented midway down the page in the Getting Started section. Components

All visual representations of data are distingued as five categories,referenced as Components.

  • Chart Component
    • Just as it sounds.Pie, Bar Charts, Etc.
  • Form Component
    • User input forms.Text,Number,& Dropdowns.
  • Guage Component
    • Measuring one piece of data.An example would be 'Total Monthly Sales';
  • KPI Component
    • "Key Performance Indicators" Show current values (not very visual) of multiple important metrics.
  • Table Component
    • As expected, a nicely formatted table. Could be used to show record in a database.
